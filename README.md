# Weather App - Front End

## Description

This is the repo to hold the code for the Frontend part of the weather app I'm building.
The idea behind this was to practice building a frontend from scratch using no frameworks to polish my HTML, CSS and JavaScript skills, then connecting it with a backend built with NodeJS that fetches APIs for weather, images and extras.
It's not that the world lacks Weather apps, but it's a project that I can easily find resources (APIs) to work with and it's something I wanted to build since I finished the Udacity NanoDegree, but an improved version of the one they had me build.
